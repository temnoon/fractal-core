A pulse system is a clock and a signing key. The clock value T is a deterministic, monotonic integer derived from a published anchor and tick unit. Each minted pulse pairs T with a fresh prime P drawn uniformly from a 2120-wide window and confirmed by Baillie–PSW. fractal-core ships four canonical systems below; the same protocol can be operated by anyone with their own anchor and key.

Disclosure. Code authored with the assistance of language models. The API itself and this interface use no AI agent at runtime — only deterministic prime mathematics and Ed25519 signing. Each system's parameters (anchor, tick unit, window, signing key) are published at /api/v1/pulse/{system}/parameters; signed pulses can be verified offline with the operator's public key.
